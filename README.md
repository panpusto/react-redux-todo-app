# ToDo App
**This app allows you to oganize your daily task.**
## Technologies:
- **React**
- **Redux**
- **Bootstrap 5**
- **node.js**
## How to use it:
1. Install node.js and npm.
2. Clone this repository.
3. `npm install`
4. Use node.js v.16 `nvm use 16`
5. `npm start`
6. Open your browser and go to http://127.0.0.1:3001/
